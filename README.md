<img height="300px" src="demo.png">
<img height="300px" src="demo-expanded.png">

## Features:
* Add/Edit/Delete a note with:
  * Priority
    * Blocker
    * Critical
    * Major
    * Minor
    * None
  * Reminder
  * Background color
  * Categories
    * Add/Edit/Delete categories

The tasks are sorted by priority (being Blocker at the top of the list).
